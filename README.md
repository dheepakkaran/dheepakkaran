**`📁 fairshare-wifi/`** &nbsp;—&nbsp; *Contextual Bandit for Adaptive Packet Scheduling*

> SimPy network simulation using **D-KernelUCB** (RBF kernel + discounted regression) to dynamically select between FIFO, Priority, and WFQ schedulers. Optimizes a composite reward of **70% Jain's Fairness Index + 30% normalized throughput**. Differentiates from ReinWiFi which assumes known traffic profiles — ours handles unknown user behavior.
