### `// projects`

**`fairshare-wifi/`** — *Two-layer ML for bandwidth fairness in public WiFi*

A SimPy-based network simulation using a **Random Forest Classifier** to identify unknown user behavior intent, feeding into a **Q-Learning agent** that dynamically adjusts WFQ weights. Optimizes for Jain's Fairness Index rather than raw throughput — a deliberate design choice differentiating it from prior work (ReinWiFi) which assumes known traffic profiles.
